# stark-spring-jpa-demo
Spring和Jpa的集成Demo.(涉及SpringMVC+Spring+Spring Data JPA).包含缓存等扩展.下面是个人的博客文章,欢迎阅读.

## 工具和框架 (Tools and Frameworks)
- Java: JDK6
- IDE: Eclipse Indigo
- Version:  Spring 3.2.5 / Spring Data Jpa 1.5.0
- DataBase: MySql
- Web Server: Tomcat
